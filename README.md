# Student-Management CRUD Application
Student management system built using Flask, MySQL, HTML, CSS and Bootstrap

## Features

* Add student records
* Update existing records
* Delete records
* View all student entries
* Flash messages for user actions

## Technologies Used

* Python
* Flask
* MySQL
* HTML
* Bootstrap

## Project Structure

* static/
* templates/
* main.py

## How to Run

1. Install dependencies
2. Configure MySQL
3. Run main.py

## Future Improvements

* Search functionality
* Pagination
* Login authentication
